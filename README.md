# 课程管理系统 java1202

## 项目概述

课程管理系统是一款基于前后端分离架构的Web应用，旨在提供高效便捷的课程管理功能。系统结合了Java和SpringBoot后端技术，以及Vue前端框架，确保了系统的高效性和稳定性。

## 技术栈

- **后端：** Java、SpringBoot、Mysql
- **前端：** Vue
- **开发工具：** IDEA、JDK1.8、Maven

## 系统角色

系统定义了以下角色：

- 管理员
- 教师
- 学生

## 功能模块

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 系统环境

- **运行环境：** Windows

## 部署步骤

1. 搭建开发环境，包括IDEA、JDK1.8、Maven等。
2. 配置数据库Mysql8.0。
3. 导入数据库脚本。
4. 编译并运行后端服务。
5. 前端项目使用Vue开发工具运行。
